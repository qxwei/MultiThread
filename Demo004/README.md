Demo004 ReentrantLock的使
========
* **app:** ReentrantLock同步
* **app1:** ReentrantLock的等待通知 signal、await
* **app2:** ReentrantLock多条件通知等待
* **app3:** 异常锁释(抛出异常线程结束)
* **app4:** synchronized 代码块同步，一些异步，一些同
* **app5:** synchronized static  class  object  死锁一
* **app6:** 静态内部类
* **app7:** volatile 使变量在多个线程间可


* **NOTE:** 
> 方法内的变量是线程安全的  
> 实例变量是非线程安全 
> synchronized不具有继承 
> 提高效率尽量缩小synchronized的范
> String常量指向同一对象
