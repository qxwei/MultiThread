Demo003 线程间通信
========
* **app1:** wait/notify 对锁的影响
* **app2:** 生产者消费者
* **app3:** 通过管道线程间通信
* **app4:** join的使用及特性
* **app5:** ThreadLocal线程私有数据
* **app6:** 静态内部类
* **app7:** volatile 使变量在多个线程间可见


* **NOTE:** 
> join 释放锁、sleep不释放锁
> join内部是由wait实现的
> synchronized不具有继承性  
> 提高效率尽量缩小synchronized的范围 
> String常量池,指向同一对象
